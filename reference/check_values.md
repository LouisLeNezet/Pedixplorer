# Check values in a slot

Check if the all the values in a slot are in a vector of values.

## Usage

``` r
check_values(val, ref, name = NULL, present = TRUE)
```

## Arguments

- val:

  A vector of values to check.

- ref:

  A vector of reference values.

- name:

  A character vector with the name of the values to check.

- present:

  A logical value indicating if the values should be present or not

## Value

A character vector with the errors if any.
