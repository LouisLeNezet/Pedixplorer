# Draw arrows

Draw arrows

## Usage

``` r
draw_point(
  x,
  y,
  ggplot_gen = FALSE,
  cex = par("lwd"),
  pch = par("pch"),
  col = par("col")
)
```

## Arguments

- ggplot_gen:

  If TRUE add the segments to the ggplot object

- col:

  Line color

## Value

Plot the arrows to the current device or add it to a ggplot object
