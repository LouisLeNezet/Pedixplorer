# List of polygonal elements

Create a list of polygonal elements with x, y coordinates and theta for
the square, circle, diamond and triangle. The number of slices in each
element can be specified.

## Usage

``` r
polygons(nslice = 1, start = 90)
```

## Arguments

- nslice:

  Number of slices in each element If nslice \> 1, the elements are
  created with
  [`polyfun()`](https://louislenezet.github.io/Pedixplorer/reference/polyfun.md).

## Value

a list of polygonal elements with x, y coordinates and theta by slice.
