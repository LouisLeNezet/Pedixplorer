# Check if the fields are present in an object slot

Check if the fields are present in an object slot

## Usage

``` r
check_slot_fd(obj, slot = NULL, fields = character())
```

## Arguments

- obj:

  An object.

- slot:

  A slot of object.

- fields:

  A character vector with the fields to check.

## Value

A character vector with the errors if any.
